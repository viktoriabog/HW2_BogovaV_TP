# HW2_BogovaV_TP

В проекте реализована система работы с рецептами и списками покупок. Пользователь может создавать ингредиенты, рецепты, диетические рецепты, формировать списки покупок и объединять их между собой.

## Установка

1. Склонировать репозиторий:

```bash
git clone https://github.com/viktoriabog/HW2_BogovaV_TP.git
```

2. Перейти в папку проекта:

```bash
cd HW2_BogovaV_TP
```

3. Установить зависимости:

```bash
pip install -r requirements.txt
```

## Использование

1. Запуск тестов:

```bash
pytest
```

2. Запуск именно файла с тестами:

```bash
pytest test_recipes.py
```
